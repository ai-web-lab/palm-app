# 02. 手相学の仕様

アプリが扱う手相学のドメイン知識。実装はこの仕様を `data/palm_rules.json` に
落としたもの。

---

## 7大線

### 基本4線（誰にでもある・診断の中心）

| 線 | テーマ | 主な判定特徴 |
|---|---|---|
| 生命線 | 生命力・健康・バイタリティ | 長さ・カーブ・濃さ |
| 知能線 | 思考タイプ・才能・適性 | 傾き・長さ |
| 感情線 | 感情・愛情・対人関係 | 長さ・起点の高さ・濃さ |
| 運命線 | 仕事運・人生の流れ | 有無・長さ・濃さ |

### extra3（ない人も多い・補助）

| 線 | テーマ | 主な判定特徴 |
|---|---|---|
| 太陽線 | 成功・人気・金運 | 有無・長さ・濃さ・本数 |
| 財運線 | 金運・蓄財力 | 有無・濃さ・本数 |
| 結婚線 | 結婚・出会いの時期 | 有無・長さ・濃さ・向き・本数 |

各線の具体的な解釈ルールは `data/palm_rules.json` の `lines[*].rules` を正とする。

## 左右の手の扱い

### 基本原則
- **左手＝先天運**（生まれ持った資質・前半生・無意識）
- **右手＝後天運**（努力で築いた今・後半生・意識的行動）
- **線の意味そのものは左右で不変。** 変わるのは時間軸ラベルだけ。

### 利き手の扱い（本アプリの採用）
- 利き手を「いまのあなた（後天運）」、非利き手を「生まれ持った本質（先天運）」とする。
- 右利きなら 右=後天 / 左=先天。左利きなら 左=後天 / 右=先天。

> ※ 流派により「男左女右」など性別で見る説もあるが、**本アプリは利き手方式のみ採用**。
> 性別では意味を変えない（調査の結果、現代の主流では男女で見方は変わらないため）。

### 撮影方針
- 必ず**片手ずつ**撮影する。両手モードでも右→左と1枚ずつ。
- 片手モードなら、その手を主として診断。
- 両手モードなら、利き手を主に診断し、加えて**左右差**を読む（後フェーズ）。

### 左右差（両手モードの付加価値・後フェーズ）
- 左右がほぼ同じ → 運命通りに自分らしく生きている。
- 左右が大きく違う → 先天と後天にギャップ、変化や波が起こりやすい。
- 右が左より良い → 素質を努力で活かし成長してきた。

## 流年法（時期判定・後フェーズ）

線上の位置を年齢に対応させる。線ごとに目盛りが異なる。

- **運命線**：手首=若年 〜 中指付け根=老年。途切れの位置の年齢が転機。
- **結婚線**：感情線寄り=早婚 〜 小指付け根寄り=晩婚。中間が30代前半ごろ。

> 目盛りは流派で割れるため、アプリは1つ採用し「30代前半ごろ」のように
> **幅をもった表現**にする。ピンポイントの年齢断定はしない。

## 診断トーンの方針（最重要）

### 禁止
- 寿命・死の断定/示唆（「生命線が短い＝短命」は誤りであり禁止）
- 重い病気の断定
- 人格否定
- 結婚/離婚/金運の断定

### 推奨（ここまではOK）
- 健康への気遣い：「休息を意識して」「体調の変化に目を向けて」
- 人間関係の諭し：「少し慎重に」「ひと呼吸おくと吉」
- 前向きな言い換え：線が「ない」も個性として読む

### 基準
対面の占い師が、相手を傷つけず前向きな気づきを促すトーン。

## 深掘り解説とアドバイスの考え方

単一特徴の解説だけでなく、**特徴の組み合わせ**で踏み込んだ読みを出す。

- 例）生命線が「濃い×長い」→「心身ともにスタミナが豊富。多少の無理も乗り越えられる」
- 例）感情線が「長い×濃い」→「愛情深く情熱的。一方で思い入れが強くなりすぎることも」

さらに、特徴に応じた**ひとことアドバイス**（諭し）を添える。
これにより「自分だけの占い」感と、占い館的な手触りを出す。

データ構造は `lines[*].combos`（組み合わせ→解説）と `lines[*].advice`（特徴→諭し）。
詳細は `docs/06_data_schema.md`。
