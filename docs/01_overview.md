# 01. 企画概要

## 何を作るか

カメラで撮影した手のひらから手相の線を検出し、本格的な手相学にもとづいて
占うWebアプリ。ブラウザで動作する。

## 誰のためか

- 手相に興味はあるが、線の名前や見方を知らない一般ユーザー。
- 「自分で線を選ぶ」のではなく「撮るだけで占ってもらえる」体験を求める層。

→ このため、ユーザーに線の知識を要求しない設計にする（撮影→自動判定→解説）。

## 提供価値

- 撮るだけで、自分の手相が**写真の上に可視化**される。
- 検出された線ごとに、長さ・濃さ・形に応じた**自分だけの診断**が読める。
- 前向きな**気づき・アドバイス**が得られる（占い館の対面体験に近い手触り）。

## スコープ

### 入る（MVP〜）
- 利き手の選択
- 占う手の選択（右 / 左 / 両手）
- 手のひらの撮影（片手ずつ）
- 基本4線（生命線・知能線・感情線・運命線）の検出と診断
- 写真への線オーバーレイ + タップで解説
- 特徴の組み合わせによる深掘り解説 + ひとことアドバイス

### 後フェーズ
- extra3線（太陽線・財運線・結婚線）
- 両手の左右差診断
- 運命線・結婚線の流年法（時期判定）
- 8つの丘、特殊線（ますかけ線など）

### 入れない
- 寿命・病気・死の断定的診断（恒久的に非対象）
- 対面鑑定の代替を謳うこと（あくまで娯楽）

## 用語

| 用語 | 意味 |
|---|---|
| 基本4線 | 生命線・知能線・感情線・運命線。誰にでもあり診断の中心 |
| 7大線 | 基本4線 + 太陽線・財運線・結婚線 |
| extra3 | 7大線のうち太陽線・財運線・結婚線。ない人も多い |
| 先天運 | 生まれ持った資質。左手が表す |
| 後天運 | 努力で築いた今の自分。右手が表す |
| 流年法 | 線上の位置を年齢に対応させ、時期を読む手法 |
| 特徴量 | 線の長さ・濃さ・カーブ等、診断の入力となる値 |

## 非機能要件

- モバイル前提（スマホのカメラで撮る）。レスポンシブ必須。
- プライバシー：手の画像はできるだけ端末内で処理する。
- 娯楽目的の明記を結果画面に常設する。
