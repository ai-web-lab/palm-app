# 09. 手型・指の特徴による読み取り設計（AIなし・端末内・正直）

既定（AIなし）の正確性・正直さを高めるための設計。ピクセル精度の線トレース（撤去済み）に
頼らず、**MediaPipe の21ランドマークから確実に測れる量だけ**を使って読み取る。

方針は A→B：
- **A（本書）**：手型（四大分類）＋指の特徴（landmark由来＝最も確実）。
- **B（次段）**：基本4線の「濃さ・presence・目立ち度」だけを実測（座標・オーバーレイは復活させない）。

> 正直さの原則：**測れたものだけ言う**。確信度が低い特徴は `standard`（穏やか）にして断定しない。
> 病気・健康・寿命には触れない（CLAUDE.md／娯楽目的）。指の長短は性格傾向の“遊び”として扱う。

## A-1. 手型の四大分類（根拠あり）

手のひらの形（正方形/長方形）×指の長さ（短い/長い）の2×2で4タイプ。西洋手相のエレメント分類。

| 手のひら＼指 | 指が短い | 指が長い |
|---|---|---|
| **正方形**（縦≒横） | **地の手**（Earth） | **風の手**（Air） |
| **長方形**（縦長） | **火の手**（Fire） | **水の手**（Water） |

- **地の手**：現実的・勤勉・堅実・粘り強い・落ち着き。職人／経営者に多い。
- **火の手**：情熱的・行動的・フットワーク軽い・リーダーシップ。明るく前向き。
- **風の手**：柔軟・知的・好奇心旺盛・変化と刺激を好む・コミュ力。
- **水の手**：繊細・優しい・感受性豊か・気配り上手・想像力。

出典：ヨムーノ手相鑑定士監修 / note(Lisa) / Spicomi / 占いTVニュース（本書末尾リンク）。

## A-2. 指の特徴（各指の意味・根拠あり）

| 指 | テーマ | 長い（＝目安を超える） | 短い |
|---|---|---|---|
| 親指(thumb) | 意志・生命力 | しっかり＝活力・意志が強い | 控えめ・マイペース |
| 人差し指(index) | 感受性・自我・リーダー性 | 感受性豊か・自尊心・指導力 | 奥手・控えめ・縁の下 |
| 中指(middle) | 知性・慎重さ | 論理的・計画的・慎重 | 行動派・考えるより動く |
| 薬指(ring) | 愛情・センス・表現・人気 | 表現力・華・勝負運 | 堅実・控えめな自己表現 |
| 小指(pinky) | 社交性・コミュニケーション | 社交的・話術・商才 | 一匹狼・単独行動を好む |
| 全体傾向 | — | 指が長い＝創造的・芸術肌 | 指が短い＝実用的・粘り強い |

出典：JAAMP / JFTA / SARA / zired（本書末尾リンク）。

## A-3. MediaPipe ランドマークでの測定式

ランドマーク番号：0=手首、親指1-4、人差し指5-8、中指9-12、薬指13-16、小指17-20
（各指：MCP付け根→PIP→DIP→TIP先端）。距離の**比率**を使うので拡大縮小に不変。

### 手のひらの形（縦横比）
- `palmLen = dist(0, mid(5,17))` … 手首→指付け根ラインの中点（縦）
- `palmBrd = dist(5, 17)` … 人差し指付け根→小指付け根（横）
- `palmRatio = palmLen / palmBrd`
- 判定：`palmRatio` が小さい＝正方形寄り、大きい＝長方形（縦長）寄り。
  **閾値は実写サンプルで較正**（暫定：< ~1.35 を正方形、以上を長方形。要検証）。

### 指の長さ（手のひらに対する相対）
- `middleLen = dist(9, 12)` … 中指の付け根→先端
- `fingerRatio = middleLen / palmLen`
- 判定：`fingerRatio` が大きい＝指が長い。**閾値は較正**（暫定：> ~0.72 を長い。要検証）。

### 個別の指（**相対比較＝最も確実**：閾値較正が要らない）
- 人差し指 vs 薬指：`dist(5,8)` と `dist(13,16)` の比較。
  - 人差し指>薬指 → 「人差し指が長め」＝自我・リーダー性。
  - 薬指>人差し指 → 「薬指が長め」＝表現力・勝負運。
- 小指の長さ：小指先端(20)が薬指の第一関節(DIP=15)を**超えるか**で長短判定（古典的な見方）。
- 親指の開き：ベクトル(2→4)と(5→8)のなす角、または(1→4)と手のひら軸の角度＝柔軟性の目安。

### 補助（pose依存＝低信頼、確信時のみ）
- 指の開き（隣接指先の間隔／手サイズ）、指の傾き（各指のMCP→TIP方向）、関節のふしの目立ち度。
  → 指を開く/閉じるで大きく変わるため、**基本は使わないか弱く**扱う。

## A-4. 信頼度の階層（正直さの担保）

| 測定 | 信頼度 | 扱い |
|---|---|---|
| 相対比較（人差し指vs薬指／小指vs薬指関節） | ◎ pose・スケール不変 | 断定してよい |
| 手型（palmRatio×fingerRatio） | ○ 較正した閾値で | 断定。境界付近は「〜寄り」に |
| 個別指の絶対長・親指角 | △ 透視・poseで揺れる | 確信時のみ |
| 指の開き・傾き・ふし | △〜✕ pose依存大 | 原則不使用／弱く |

- ランドマークが取れない／手が正対していない場合は**測定を諦め**、`standard`（穏やか）で返す。
  「うまく読み取れませんでした」を正直に出し、当てずっぽうを出さない。

## A-5. 知識ベース（palm_rules.json）拡張計画

## A-6. 実装方針（指示後）

1. `lib/handMetrics.ts`（新規・純粋関数）：ランドマーク配列 → {handType, fingers, confidence}。
   距離・比率・相対比較のみ。テスト可能（合成ランドマークで単体テスト）。
2. `lib/handDetect.ts` の `getLandmarker`（温存済み）で 21点を取得 → handMetrics へ。
3. `genMockFeatures` を **実測 read** に置換（`app/page.tsx`）。取得失敗時は正直に穏やか表示。
4. `ResultStep` に手型の見出し＋指の補足を追加（線オーバーレイは復活させない）。
5. B：`extractFromGray`（温存）を「濃さ/presence/目立ち度」だけに使い、基本4線の depth/presence を実測で埋める。
